Флюсы высокотемпературные

Высокотемпературными считаются паяльные флюсы с максимальной температурой активности более 450 ⁰C.

По составу основных компонентов высокотемпературные паяльные флюсы принято делить на:

  • паяльные флюсы на основе буры и соединений борной кислоты,
  • паяльные флюсы на основе хлористых солей,
  • паяльные флюсы на основе соединений борной кислоты и хлористых солей.

Большинство давно используемых флюсов для пайки производятся путём смешения компонентов. Используемые вещества, как правило, продукты массового производства технического назначения.

Современные флюсы для высокотемпературной пайки (паяльные флюсы для твёрдой пайки) имеют сложное строение, многие из них изготавливаются путём химического синтеза.

Для выбора паяльного флюса, подходящего для конкретной задачи пайки, можно воспользоваться таблицей подбора флюсов. Выбрать флюсы для пайки, подходящие для использования с выбранным припоем и спаиваемыми материалами. Далее, по описанию каждого из паяльных флюсов, определить подходящий флюс по методу нанесения и методу нагрева под пайку. Метод удаления остатков паяльного флюса, так же может быть важен.
