# Сайт представляющий возможность ознакомиться с продукцией компании и заказать её онлайн

Корзина работает локально, через local.storage браузера
javaScript собираеться в один файл системой сборки webpack, что позволяет разбить программный код на модули, но на страницу подключать один файл javaScript

После оформления заказа через сервис emailJS заявка приходит на почту
